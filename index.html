Day 2

松島 + 青森弘前

1月5日（日）· 仙台出發

上午
交通仙台 → 松島
從仙台搭乘東北本線至松島海岸站，約40分鐘。
上午–中午
美食松島牡蠣吃到飽
松島為日本三景之一，冬季（1月）正值牡蠣盛產期，當地多間餐廳提供牡蠣吃到飽（かき食べ放題）。
提醒冬季1月份牡蠣季進入尾聲，建議提前確認餐廳營業並預約
  • 松島さかな市場周邊有多家提供牡蠣烤吃到飽的攤位
  • 烤牡蠣（焼き牡蠣）、牡蠣飯、牡蠣湯都值得嘗試
下午
交通前往青森弘前
松島 → 仙台搭乘新幹線 → 新青森，再轉 JR 奧羽本線至弘前站。全程約2.5～3小時。
此日行程跨距較大，建議安排松島上午、弘前下午到傍晚
下午–晚上
觀光弘前蘋果派地圖巡禮
弘前市有「弘前アップルパイ研究会」，集結全市數十間各具特色的蘋果派店，可索取地圖或上網查詢路線。
  • 必吃：各店蘋果派風格差異大，酥皮、奶酥、法式等皆有
  • 伴手禮建議選購弘前產蘋果果凍或蘋果醬
地圖
傍晚
觀光睡魔之家 WA・RASSE
位於弘前車站附近的青森睡魔祭（ねぶた）資料館，展示真實等比例的睡魔花車，即使非祭典期間也可體驗。
開放時間9:00–19:00（因時期而異），建議出發前確認
傍晚
觀光青森縣立美術館
收藏奈良美智等知名藝術家作品，建築本身亦是看點。距弘前車站需搭乘巴士至青森市，若時間有限可擇一參觀。
弘前與青森市之間約40分鐘車程，若時間不夠可改只逛弘前市區
晚上
交通返回仙台
搭乘新幹線回仙台，回 APA 飯店。
Day 3

銀山溫泉一日遊

1月6日（一）· 仙台出發

全天
行程KKday 銀山溫泉一日遊
銀山溫泉（銀山温泉）位於山形縣，為日本最具代表性的大正浪漫溫泉街。冬季積雪景色如明信片，夜晚燈籠倒映河面極為夢幻。
  • 交通：從仙台出發需約2~2.5小時，KKday 行程通常含接送
  • 特色：木造旅館林立的溫泉街、銀山川旁散策
  • 注意：非住宿者也可在日歸泡湯設施泡湯
  • 購物：溫泉饅頭、玉こんにゃく為當地名物
建議提前於 KKday 預訂行程，1月冬季為旺季
晚上
住宿返回仙台 APA
今晚為仙台最後一晚，明日移動至函館。建議整理行李。
Day 4

移動函館 · 夜景

1月7日（二）· 仙台 → 函館

上午
交通仙台 → 函館
搭乘新幹線 はやぶさ（東北新幹線 + 北海道新幹線），仙台 → 新函館北斗，約1小時50分。再轉 JR 至函館站，約20分。
交通費建議使用 JR Pass（北海道・東日本パス）可大幅節省費用
下午
觀光五稜郭
日本唯一的五芒星（西式）城堡遺址，冬季積雪呈現壯觀白色星形。可搭乘五稜郭塔電梯俯瞰全景。
  • 五稜郭塔：¥900，推薦在雪景中俯瞰
  • 冬季函館平均氣溫-5°C，注意保暖
地圖
傍晚–晚上
觀光函館山夜景
函館山夜景為日本三大夜景之一，冬季空氣清澈視野絕佳。建議傍晚搭纜車上山（約3分鐘），欣賞函館市區漂亮的腰形輪廓夜景。
纜車來回 ¥1,500 / 冬季因天氣可能停駛，可改搭巴士

Day 5

移動札幌 · 帝王蟹

1月8日（三）· 函館 → 札幌

上午
美食函館朝市
函館朝市開門時間為清晨5點，是北海道最著名的早市。海鮮丼、鮭魚卵、帆立貝等新鮮海產一應俱全。
推薦現撈海鮮丼、函館烏賊（夏季限定需確認）
上午
交通函館 → 札幌
搭乘 JR 特急 北斗號（函館 → 札幌），約3.5小時。沿途可欣賞北海道大雪原景色。
北海道新幹線延伸至札幌預計2030年，目前需搭特急列車
下午
購物狸小路商店街
狸小路（狸小路商店街）為札幌最熱鬧的室內商店街，共7丁目，全長約900公尺，藥妝、餐廳、土產齊聚。
地圖
晚上
美食エビかに合戦 帝王蟹吃到飽
エビかに合戦（蝦蟹合戰）是北海道知名的海鮮吃到飽餐廳，帝王蟹、毛蟹、雪場蟹等豐盛海鮮應有盡有。
  • 地址：札幌市中央區（狸小路附近有分店）
  • 強烈建議提前訂位，旺季可能一位難求
  • 吃到飽時間通常為90~120分鐘
提醒人均費用約 ¥5,000–8,000，確認是否含飲料

郊外一日遊

1月9日（四）· 札幌出發

全天
行程KKday 一日遊（待選擇）
以下為推薦選項，可於 KKday 搜尋相關行程：
  • 洞爺湖：火山口湖，冬季結冰景色壯觀，有溫泉旅館，距札幌約2小時
  • 支笏湖：日本最北的不凍湖，冬季有「支笏湖冰濤祭」（1月下旬），距札幌約1小時，交通較方便
  • 定山渓溫泉：距札幌約1小時的近郊溫泉，適合半日遊
  • 白色戀人公園：在市區內，雨天備案
建議支笏湖因距離近且1月有冰濤祭前置活動，可列為優先
Day 7

小樽

1月10日（五）· 札幌出發

上午
交通札幌 → 小樽
搭乘 JR 函館本線，札幌 → 小樽約35分鐘，班次頻繁。
上午–下午
觀光小樽運河
小樽運河是北海道最具代表性的風景，冬季河畔積雪、煤氣燈倒映水面，是攝影勝地。沿運河有咖啡廳、餐廳及玻璃工藝品店。
地圖
下午
購物堺町通 · 玻璃工藝
小樽以玻璃工藝聞名，堺町通上有北一硝子（Kitaichi Glass）等知名玻璃店，可購買手工玻璃製品作為伴手禮。
  • 北一硝子：最大玻璃工藝店，附設煤油燈咖啡館
  • 大正硝子：另一知名玻璃工藝品牌
午餐 / 晚餐
美食壽司通
小樽有「壽司屋通り」，聚集多間高質素的握壽司餐廳，使用北海道當季海鮮食材。
推薦政壽司、おたる政寿司、小樽 きたかろ等，建議午餐時段前往（人較少）
傍晚
交通返回札幌
搭乘 JR 返回札幌，整理行李準備隔天回台。

旅遊提醒
仙台 · 北海道冬季氣候
+
  • 仙台 1月：平均氣溫 -2°C ～ 3°C，偶有降雪但積雪量不如北海道
  • 函館 1月：平均氣溫 -8°C ～ -1°C，風大、積雪明顯
  • 札幌 1月：平均氣溫 -10°C ～ -3°C，為全旅程最寒冷城市
  • 市區街道可能有結冰，穿防滑雪靴是必要的
  • 日本室內暖氣充足，進出室內需注意溫差
交通 · JR Pass 建議
+
  • JR 北海道・東日本パス：可搭乘 JR 東日本（仙台地區）+ JR 北海道（函館、札幌）的特急、新幹線，連續5天使用
  • 此行從仙台→函館→札幌全程用 JR，建議購買此 pass
  • 需在台灣出發前於 JR 官網或 KKday 購買兌換券，抵日後兌換
  • Suica / ICOCA：用於市區地鐵、巴士，可在仙台、函館、札幌的 JR 站購買
1/5 行程注意（松島 + 弘前同天）
+
  • 松島和弘前同一天行程跨距較大，建議松島上午不要拖太久（安排2～3小時）
  • 弘前與青森市有段距離，若想同時看睡魔資料館（在青森市）與弘前蘋果派，需要精確掌握時間
  • 可考慮只去弘前（蘋果派 + 弘前城附近），省略青森縣立美術館，另擇時間
  • 東北冬季天黑早（下午4點即入夜），注意時間規劃
飲食 · 訂位建議
+
  • 善治郎牛舌（仙台）：旺季排隊長，建議開店前到場或使用 Google Map 確認是否可線上預約
  • エビかに合戦（札幌）：6人用餐建議至少1個月前訂位，可透過 Tabelog 或官網預約
  • 小樽壽司通餐廳：中午時段比晚餐等待時間短，午餐前往較佳
  • 北海道海鮮整體物價不低，吃到飽行程較划算
購物 · 伴手禮建議
+
  • 仙台：萩の月（最著名仙台銘菓）、牛舌真空包裝
  • 弘前：蘋果派、蘋果酒、蘋果果凍
  • 函館：いかめし（魷魚飯）真空包、昆布加工品
  • 札幌 / 小樽：白色戀人餅乾、六花亭甜點、北海道牛乳糖
  • 大型藥妝品牌：狸小路的 Don Quijote（唐吉訶德）24小時營業，適合最後一晚掃貨
